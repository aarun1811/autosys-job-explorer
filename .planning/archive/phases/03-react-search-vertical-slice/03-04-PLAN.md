<objective>
Three things this plan ships, all enabling Plan 05 SearchGrid to be config-driven:

1. **`configCategoryToColDefs` adapter** — translates a `CategoryConfigV4` (parsed from `/api/v4/search/config`) into an array of AG-Grid `ColDef`s, resolving `cellRenderer` string keys through the registry, converting `cellStyle` kebab-case keys to camelCase, and gracefully handling unknown renderer keys. This is the SINGLE place that touches the JSON-vs-AG-Grid impedance — the rest of SearchGrid stays declarative.

2. **AG-Grid Enterprise module registration in `main.tsx`** — add `ExcelExportModule`, `ColumnsToolPanelModule`, `FiltersToolPanelModule` to the bootstrap so `gridApi.exportDataAsExcel()` (Plan 06) and `sideBar.toolPanels` (Plan 05) don't throw "module not registered" at runtime (Pitfall 9).

3. **shadcn primitive vendoring** — `input`, `badge`, `separator`, `command`, `popover`, `tooltip`, `dropdown-menu`, `skeleton` at pin 3.8.5. Plans 05 (SearchGrid sideBar UI), 06 (SearchBar + SearchToolbar), 07 (SearchPage) consume these.

Purpose: SEARCH-01 (config-driven search) hinges on this adapter — it is the operationalization of the memory-locked principle ("adding a column = JSON edit + restart, not React change"). The two side-tasks (modules + shadcn vendoring) gate Plans 05/06 from runtime explosions and component-not-found errors.

    ```
    Imports: ColDef from 'ag-grid-community'; CategoryConfigV4, ColumnDefinitionV4 from '@/search/types'; cellRenderers from '@/search/renderers/registry'.

    function toCamelCaseStyle(style: Record<string, string>): Record<string, string> {
      iterate Object.entries(style); for each [k, v], compute camel = k.replace(/-([a-z])/g, (_, c) => c.toUpperCase()); set out[camel] = v; return out.
    }

    export function configCategoryToColDefs(cat: CategoryConfigV4): ColDef[]:
      return cat.columns.map((c: ColumnDefinitionV4): ColDef => ({
        field: c.field,
        headerName: c.headerName,
        width: c.width,
        hide: c.hide ?? false,
        sortable: c.sortable ?? true,
        filter: c.filter === false ? false : 'agTextColumnFilter',   // boolean→true means default text filter; false means no filter
        resizable: c.resizable ?? true,
        rowGroup: c.rowGroup ?? false,
        pinned: c.pinned ?? undefined,
        cellRenderer: c.cellRenderer ? cellRenderers[c.cellRenderer] ?? undefined : undefined,
        cellRendererParams: c.cellRendererParams,
        cellStyle: c.cellStyle ? toCamelCaseStyle(c.cellStyle) : undefined,
      }))
    ```
