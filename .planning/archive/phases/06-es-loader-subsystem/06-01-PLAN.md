Bootstrap the local-dev seed with the three pieces of state Phase 6 requires before any backend code can boot: (1) the ShedLock `shedlock` table in the rectrace schema; (2) the `loader_run_history` table in the rectrace schema; (3) the `rectrace_core_alias` ES alias on the existing `rectrace_core_index`.

The `LoaderRunHistoryService` inserts on every run — the table must exist or every run errors on its first statement.
