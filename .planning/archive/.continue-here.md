## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Stale-worktree commits to main checkout | Multiple executor agents had worktrees created from old commits (90d22c9 pre-modernization). When they used absolute paths to repo files, edits landed on the main checkout's milestone/modernization branch instead of their worktree branch. Two early ones also ran `git symbolic-ref HEAD` on main. | advisory | Always include the `MANDATORY_FIRST_STEP` `git reset --hard milestone/modernization` block at the top of every executor prompt. Verify worktree is current before any reads/writes. Don't allow `git symbolic-ref` or `git update-ref` on a protected ref. |
| AskUserQuestion blocks autonomous flow | The default `/gsd-autonomous` smart_discuss step asks grey-area questions per phase via AskUserQuestion. With the user away, this would have hung. | advisory | Set `workflow.skip_discuss=true` and `workflow.auto_advance=true` config flags for hands-off autonomous runs. ROADMAP phase goal + success criteria become the spec. |
| Planner schema inferred from Angular types instead of live response | Phase 3 RESEARCH.md Open Question #1 flagged that `InitialSearchResponseV4` shape should be verified with a live curl during planning; planner did not. Phase 3 UAT in Playwright surfaced 4 real defects (Zod schema mismatch + 5 missing AG-Grid v35 modules + ColumnApiModule + useRecentSearches no cross-instance sync). | advisory | When a research file flags "verify with live curl," do it during planning. Plan-checker should grep RESEARCH.md for `[NEEDS LIVE VERIFICATION]` markers and gate the plan on them. |
| Unscoped Maven Enforcer dependencyConvergence | Phase 7 Plan 05 first attempt used unscoped `<dependencyConvergence/>` which fails on a pre-existing httpclient 4.5.13 vs 4.5.14 split in spring-data-elasticsearch transitive tree (orthogonal to OBS-08). | advisory | Use `requireSameVersions` scoped to the artifacts you actually care about; preserve the "two-rule defense" intent without policing unrelated dep trees. |
