    `apiFetch(path, init?)`:
    - Generate: `const correlationId = crypto.randomUUID().replace(/-/g, '')`
    - Fetch: `fetch(path, { ...init, headers: { 'Content-Type': 'application/json', ...init?.headers, 'X-Correlation-Id': correlationId } })`
    - On `!res.ok`: `throw Object.assign(new Error(\`HTTP ${res.status} ${path}\`), { correlationId })`
    - Return `res` on success (caller calls `.json()`)

    `queryClient` with `QueryCache.onError`:
    - Read `.correlationId` from the error: `const corrId = (error as { correlationId?: string }).correlationId`
    - Fire: `toast.error('Request failed', { description: corrId ? \`Error reference: ${corrId}\` : 'Something went wrong. Check the browser console for details.' })`
    - Default options: `staleTime: 5 * 60 * 1000`, `gcTime: 30 * 60 * 1000`, `retry: 1`, `refetchOnWindowFocus: false`
